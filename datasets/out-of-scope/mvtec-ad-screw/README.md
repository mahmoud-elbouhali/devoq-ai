# MVTec AD Screw

Source :

- Kaggle : `thomasdubail/screwanomalies-detection`

Pourquoi ce dataset est hors sujet pour le besoin principal :

- il sert a la detection d'anomalies sur une vis unique
- il ne contient pas des scenes de plusieurs vis a compter sur une surface
- les masques `ground_truth` representent des defauts, pas des objets a compter

Ce dataset peut rester utile pour :

- comprendre la morphologie d'une vis
- un futur sous-projet d'inspection qualite

Ce dataset ne doit pas etre utilise comme dataset principal pour entrainer le comptage multi-vis.
